# 스터디 소개
- 스터디 일정 : 주 1회 온라인 화상 회의 1-2시간 매주 월요일 22시
- 인원 : 4명
- 스터디 방식 : 각자 관심 기술을 자유롭게 학습하여 팀원에게 발표
